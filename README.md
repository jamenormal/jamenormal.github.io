# chat-app-express

[[Workshop] ทำ Chat Application ด้วย Node.js, Express และ Socket.io](https://devahoy.com/chat-app-with-express-socketio/)

![Chat App](./chat-app.png)

## วิธีการติดตั้ง

ติดตั้ง dependencies

```
npm install
```

Start server

```
npm start
```

เว็บไซต์จะรันอยู่ที่ http://localhost:5555
